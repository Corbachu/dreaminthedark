AITD_PakEdit
============


Alone in the Dark PakEdit allows to modify PAK file for AITD and FITD.
It is based on yaz0r's FITD PAK reading implementation.

---------

Usage
-----
Open a PAK, then choose one of the file, export it, modify it and
re-import it.
